# Portfolio

Portfolio website built in NextJs

This was a website built to showcase my technical skills.

## Technologies Used

This project was built using these technologies.

- TypeScript
- NextJs
- NextAuth
- Mongodb
- Mongoose
- TailwindCss
- GitHub
